# Tetris3DPhysics
テトリス3D(物理)を作ってみた

# 目的
3D版のテトリスを作ること．面白そうなので物理演算させること

# 機能
- 3Dに拡張したテトリスがプレイ可能です．
- 難易度はイージー，ノーマル，ベリーハードの３つあります．
- HDRPによってきれいなライティングが実現できました．
- 一段が８割以上埋まるとその段が消えます．
  - ただし，ベリーハードは100%埋める必要があります．
